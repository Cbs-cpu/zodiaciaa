# Zodiac

A roguelite action game where you fight against enemies inspired by the 12 signs of the zodiac.

## Structure

- `scenes/` — All Godot scenes (.tscn)
- `scripts/` — All GDScript files (.gd)
- `assets/` — Sprites, sounds, and music
- `docs/` — Design and architecture documentation

## Enemies

Aries, Taurus, Gemini, Cancer, Leo, Virgo, Libra, Scorpio, Sagittarius, Capricorn, Aquarius, Pisces

## Built With

- Godot 4
